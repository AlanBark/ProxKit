# Proxy Print and Cut

A browser-based tool for generating print-and-cut PDF files compatible with Silhouette cutting machines. Upload your images, configure card layouts, and download a ready-to-cut PDF with registration marks.

## Features

- Upload and manage multiple card images
- Customizable card dimensions and bleed settings
- Automatic registration mark generation for precise cutting
- Support for both A4 and Letter paper sizes
- Live preview of card layouts
- Export print-ready PDFs with proper alignment marks

## Usage

1. Upload your card images using the file upload interface
2. Configure card settings (dimensions, bleed, spacing)
3. Adjust file settings (paper size, margins)
4. Preview your layout
5. Download the generated PDF file
